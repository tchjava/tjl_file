#### #20001 获取openId

- **请求URL**

  >/wx/user/queryopenid

- **请求方式**

  > POST

- **请求示例**

  ```json
  {
   	platform:"wxapp",//必须，就填wxapp
      code:""//小程序的code码，必须
  }
  ```
- **返回示例**

  ```json
  {
      info:{
          openId:1,
          status:"00A-已绑定,00B-未绑定"
      },
      result:{
          success:true,
          message:"请求成功"
      }
  }
  ```

- 错误码

  | success |   message    |
  | :-----: | :----------: |
  |  true   |   请求成功   |
  |  false  | 具体错误信息 |
